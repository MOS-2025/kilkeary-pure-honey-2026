# kilkeary-pure-honey-2026
**Kilkeary Pure Honey**

**Kilkeary Pure Honey** is a responsive, static website built with **HTML5**, **Bootstrap 5**, and **custom CSS** to showcase a family-run Irish honey producer.  
The site highlights the brand story, honey varieties, beekeeping classes, media content, and contact information in a clean, modern layout.
